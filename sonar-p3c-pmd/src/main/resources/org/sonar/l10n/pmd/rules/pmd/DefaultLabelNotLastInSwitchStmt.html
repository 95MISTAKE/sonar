Switch statements should have a default label. Example : 
public class Foo {
 void bar(int a) {
  switch (a) {
   case 1:  // do something
      break;
   default:  // the default case should be last, by convention
      break;
   case 2:
      break;
  }
 }
}

This rule is deprecated, use {rule:squid:SwitchLastCaseIsDefaultCheck} instead.
